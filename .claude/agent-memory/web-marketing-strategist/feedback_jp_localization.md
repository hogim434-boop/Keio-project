## 직역체 일본어 절대 금지
- 한국어를 그대로 옮긴 일본어 카피 (「〜であります」 「〜してください」 일변도) 는 즉시 실패.
- **Why**: 케이오 학생은 자기 부족 안에서만 통하는 슬랭 (「塾生」「ぼっち」「ガチ」「萎え」「~過ぎる」「あーね」) 을 쓴다. 비인사이더 카피는 1초 안에 들킨다.
- **How to apply**: 카피 작성 후 반드시 현지 학생 슬랭이 들어갔는지, 인스타에서 케이오 학생이 실제 쓰는 표현인지 자문. 의심되면 해당 해시태그(`#塾生` `#慶應あるある`)에서 실제 사용례 5개 이상 검색해 톤 매칭.

## 메시지 우선순위: 신뢰 → 가치 → 차별 → 비용
- 일본 시장 표준 순서. 한국식 "재미·익명 우선" 은 일본에서 거부 트리거.
- **Why**: ひま部 사건 등 학습된 불신 + 일본 소비자의 신중한 첫 접점 행동.
- **How to apply**: 어떤 광고·랜딩·DM 도 첫 메시지가 「안전한가?」 의 답이어야 함. 「재밌어 보인다」 가 첫 메시지면 거의 확정 실패.

## 인플루언서 컨택은 반드시 「謝礼明示 + 옵션 3개 제시」
- 「無料로 부탁」은 일본에서 강한 거부.
- **Why**: 일본인은 비대칭한 부탁에 답장 자체를 안 한다. 거절할 옵션이 있을 때만 회신율 높아짐.
- **How to apply**: 마이크로 인플루언서(1k~10k) 단가 표준은 팔로워 1k당 ¥3,000~5,000. 컨택 DM 에 (1) 유료 협업 (2) 무료 초대 + 후기 (3) 기타 협의 — 3옵션을 명시.

## "中の人" 운영자 표현 사용
- 운영자 정체를 완전 익명화하지 말고 「中の人」 (인사이더) 표현으로 친근감 + 익명 양쪽 충족.
- **Why**: 일본 SNS 운영 문화에서 「中の人」 는 운영자에 대한 친밀도 표현. 완전 무인격 운영 톤은 일본에서 차갑다고 느껴짐.
- **How to apply**: DM·라이브·스토리에서 「運営の中の人です」 같은 자기 호칭 사용. 단, 본명·얼굴 노출은 강제 아님.

## 시뮬레이션 콘텐츠 표기 의무
- 런치 초기 시드 글이 부족할 때 운영자가 가공한 익명 고백 콘텐츠를 만들 경우, 「※ 仮想的なエピソードです」 작은 글씨라도 반드시 표기.
- **Why**: 일본 시장은 가짜 사용자 사칭 발각 시 즉시 사망. 정직성 신호가 한국 시장보다 훨씬 무겁게 작동.
- **How to apply**: 모든 P2 (匿名告白) 콘텐츠의 마지막 슬라이드 또는 캡션 끝에 표기 일관 적용.
