## W1·W2 만 100% 완성, W3·W4 는 70% 틀로 남긴다

런치 4주 콘텐츠 캘린더를 작성할 때, 4주 전체를 사전에 완성된 카피·해시태그까지 100% 적재하지 말 것.

**Why**: 일본 시장에서는 W1 출시 후 첫 7~10일 데이터로 톤이 의외로 보정되어야 하는 경우가 많다. 가공 시드 콘텐츠가 예상보다 차갑게 받아들여지거나 (「※仮想」 표기에도 불구하고 가짜 인상), 특정 페르소나 (예: 内部進学) 가 예상보다 적게 반응할 수 있다. W3·W4 의 캡션을 W1 시점에 미리 완성하면 W2 데이터를 반영할 기회가 사라진다.

**How to apply**:
- W1·W2: 캡션·해시태그·첫 댓글까지 100% 완성, 운영자가 즉시 복붙 가능
- W3·W4: 비주얼 컨셉·포맷·필러·CTA 는 100% 결정. 단 캡션은 「틀 + ★ W2 인기 댓글 인용」 같은 placeholder 로 남길 것
- 운영자에게 「W3·W4 캡션은 W2 데이터 보고 채우는 것이 정답」 명시 필수
- `docs/marketing-content.md` (2026-05-07 작성) 가 이 원칙의 적용 사례
