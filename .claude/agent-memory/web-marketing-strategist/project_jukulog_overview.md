## 서비스 정체성 (변하지 않음)
- **Jukulog (塾ログ)** = 케이오대(慶應義塾大学) 재학생만 가입 가능한 클로즈드 익명 게시판. keio.jp / g.keio.ac.jp / sfc.keio.ac.jp 도메인 인증으로만 진입.
- "塾" 은 慶應義塾에서 따왔고 케이오 학생들이 자기 자신을 부르는 인사이더 단어 「塾生(じゅくせい)」 정체성에 직접 호소.
- 5개 카테고리: 学業・授業 / 就活 / 学校生活 / サークル / 雑談.

## 피벗 배경 (전략에 직접 영향)
- 원래 "강의 리뷰" 서비스였으나 PENMARK가 케이오 학부생 80% 점유로 차별화 불가 → "익명 자유 게시판" 으로 전환.
- **Why**: PENMARK 와 직접 충돌하지 말고 보완재로 포지셔닝하는 것이 정답. 슬로건 결정의 핵심 논리.
- **How to apply**: 마케팅 카피에서 PENMARK 를 적으로 두지 말고 「PENMARKで授業、Jukulogで本音。」 처럼 두 슬롯이 분리됨을 명시할 것.

## 1인 운영 제약 (모든 전략의 상한선)
- 사용자는 코딩 초보 1인 운영자.
- **Why**: 채널을 늘리거나 캠퍼스별 부계정으로 분산하면 임계 질량(critical mass) 도달 전 운영자 번아웃.
- **How to apply**: 계정 1개 + 인스타 단일 채널 + 자동화 도구(Buffer/Later, Meta 자동 응답) 적극 사용. 4주 사전 콘텐츠 적재 + 주 1일 강제 휴무가 운영 매뉴얼 기본 가드레일.

## MVP 명시 제외 기능 (콘텐츠 기획에 직접 영향)
- 이미지 첨부 X / 푸시·이메일 알림 X / 차단 기능 X.
- **Why**: 푸시 알림 부재 → 인스타 스토리·DM이 사실상 유일한 리텐션 채널. 이미지 첨부 부재 → 텍스트 기반 익명 고백 콘텐츠가 핵심 자산.
- **How to apply**: 리텐션 후킹은 "오늘의 베스트 글" 일일 스토리 노출로 운영. 콘텐츠 필러에서 P2 (匿名告白) 비중 20% 이상 유지.

## 메인 슬로건 결정 (확정됨, 2026-05-07)
- **「塾生だけが、本音で話せる場所。」** 채택.
- **Why**: 「塾生」은 케이오 인사이더만 쓰는 단어 → 첫 한 마디부터 부족 신호 발신. 「本音」 (본심) 은 일본 시장 특유의 무게를 가진 키워드.
- **How to apply**: 향후 광고 카피·인플루언서 협업·이메일 헤더에 일관 사용. 변경 시 브랜드 자산 손실 큼.

## 페르소나 5인의 공통 트리거 (콘텐츠 카피의 뿌리)
- "같은 塾生인데, 친구한테는 도저히 못 할 이야기".
- **Why**: 5개 페르소나(日吉ぼっち 1년 / 三田 의식 높은 3년 / SFC 프로젝트 마니아 / 内部進学 リア充 / 文学部 院生) 모두 이 트리거를 공유. 마케팅 메시지가 갈라지지 않는 근거.
- **How to apply**: 모든 카피에서 "친구·SNS·본가 어디에도 못 할 말" → "塾生限定 익명 공간" 의 흐름을 유지.

## 협업 1순위 외부 계정 (확인됨, 2026-05)
- `@mitasai_pr` (三田祭 公式 인스타, 약 6,897 팔로워).
- **Why**: 캠퍼스 가리지 않고 全 케이오 학생이 보는 유일한 공식 계정. 11월 三田祭 시즌 폭발.
- **How to apply**: 8월 말 이전 첫 DM 컨택 필수. 9월 이후엔 三田祭 준비로 후순위 밀림. 9월 카운트다운 콘텐츠 무상 제작 제공이 첫 협업 카드로 효과적.

## 일본 학생 한정 SNS 의 학습된 불신
- ひま部 (2015~2019) 사망 사건이 일본 사회에 깊이 박혀 있음 (성범죄 피해 18세 미만 SNS 2위 214건 → 강제 종료).
- **Why**: 학생 한정 SNS = 위험하다는 인식이 케이오 학생들에게 프리셋되어 있음.
- **How to apply**: 마케팅 메시지 우선순위는 1) 안전(塾生限定 인증) 40% → 2) 가치(本音) 35% → 3) 차별(캠퍼스별) 15% → 4) 비용 10%. 한국식 "재미 우선" 카피는 일본에서 즉사. 신뢰 → 가치 순서.
