---
name: 즉시 실행 스타일
description: 사용자가 선호하는 "즉시 실행 모드" — 추가 plan 없이 바로 수정 + 검증
type: feedback
---

리뷰 항목이 주어지면 추가 plan 작성 없이 즉시 코드 수정을 시작하고 npm run build까지 검증.

**Why:** 사용자가 "즉시 실행 모드. 추가 plan 작성 NO."라고 명시했음. 분석 → 계획 → 승인 요청 루프 없이 바로 실행하는 것을 원함.

**How to apply:** 수정 대상 파일과 항목이 명확한 경우 파일을 읽고 바로 수정 적용. 불명확하거나 파괴적인 변경(DB 스키마 등)만 사전에 확인 요청.
