2026-04-30 한국 대학생 타겟 두 아이디어(족보길드 / 구독N빵) 심화 분석 완료.

**최종 추천: 족보길드** (강력)

**Why (검색 결과 기반):**
- 족보길드 시장: 에브리타임 1강 독점, "기여 호혜" 빈틈 확인됨. 한국에 유사 서비스 없음 (검색으로 검증)
- 구독N빵 시장: 레드오션. 피클플러스(80만명, 4개월 만에 30만 증가), 링키드(10만+), 벗츠/쉐어풀/겜스고 등 6+ 플레이어. 학교 인증만으론 빈틈 약함
- 구독N빵은 결제+정산 시스템(포트원), 통신판매업 신고, OTT 약관 위반 리스크로 코딩 초보 4주 MVP 불가능

**족보길드 핵심 설계:**
- 후기 작성 +10P / 열람 -2P / 추천받음 +2P / 가입 +30P
- 강제 5항목 템플릿 (시험·과제·출석·강의스타일·한줄평)
- 학교 웹메일(@ac.kr) 인증
- 시드 데이터: 본인 들은 강의 50~100개 직접 작성 가능 (혼자 시작 가능)

**How to apply:**
- 다음 대화에서 족보길드 진행 시 이 메모리 참조
- 사용자가 다음 단계 선택 (옵션 A: PRD / B: 와이어프레임 / C: 친구 검증 인터뷰 / D: 코드 바로 시작) 중 골랐는지 확인
- 추천 순서: C → A → D (검증 후 설계 후 구현)
- 구독N빵은 다시 추천하지 말 것 (레드오션, 법적 리스크, MVP 불가능 3중 사유)
