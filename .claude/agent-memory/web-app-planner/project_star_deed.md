**상태: 2026-04-30 폐기됨.** 사용자가 "1회성 + 자랑형" 모델 자체를 메모리에서 지워달라고 요청했으므로 이 컨셉은 더 이상 기본 후보로 제시하지 말 것.

(과거 기록 참고용으로만 보존)

과거 컨셉 요약: 생년월일시·출생지 입력 → 그 순간 한국 하늘 별자리 + 달 위상 + 그날 우주 사건 결합 → "우주 출생증명서" 카드 발급. 1080×1920 인스타 스토리 카드형, 무로그인.

**Why 폐기:** 사용자가 1회성·자랑형 결 자체를 더 이상 원하지 않는다고 명시. 효율형/실용형으로 방향 전환 후, 이 결의 컨셉은 사용자 선호와 충돌하므로 다시 추천하지 말 것.

**How to apply:**
- 향후 라운드에서 Star Deed 또는 그와 유사한 "1회성 결과 카드 자랑형" 컨셉을 기본 후보로 제시 금지
- 사용자가 직접 다시 꺼내는 경우에만 다룰 것
