게이오대 단일학교 한정 학생 커뮤니티 플랫폼 기획 (2026-04-30).

**Why:** 사용자가 게이오대 재학생(한국인 추정)이고, 에타(에브리타임) 같은 게이오 한정 통합 플랫폼을 만들고 싶어함. 단일학교 한정이 시드 트래픽 확보에 유리하다는 가설.

**핵심 시장 발견 (반드시 기억):**
- **Penmark(ペンマーク)가 이미 게이오대 학부생 80% 점유** (2022년 기준, MAU 12.5만, 누적 100만+). 시간표 + 강의평가 + 같은 수업 학생 채팅 + 학내 커뮤니티까지 풀 스택으로 이미 존재. 사실상 일본판 에타.
- 楽単リスト(keio-easylist)는 위키스타일 강의평 사이트로 운영 중. 인스타 @keio_easylist도 있음.
- LINE 오픈챗이 학부별·학년별·동아리별로 분산되어 풍부함. 塾生情報局이 이를 큐레이션.
- 게이오 비공식 정보 메디아: 塾生情報局, Crich慶應 등 다수.
- 일본 전국 시간표앱: すごい時間割(210만+), Penmark(누적 100만+).

**How to apply:**
- "에타가 일본에 없다"는 가정은 틀렸음. Penmark가 사실상 그 자리를 차지 중.
- 게이오 한정 풀스택 클론은 후발주자로 매우 어려움. 정직하게 알려야 함.
- 대신 빈틈 가능성: (1) 한국인 유학생 한정 커뮤니티, (2) Penmark가 약한 영역(예: 익명 게시판 깊이, 중고거래, 캠퍼스간 정보 격차, 동아리 매칭), (3) SFC↔三田 캠퍼스 분리 이슈, (4) 한일 양방향 정보 (취업·인턴).
- 사용자가 한국인일 가능성 높으므로, "게이오 한국인 유학생 + 일본인 학생 다리" 포지션이 현실적 차별점.

**캠퍼스 구조:**
- 三田(미타): 문·경제·법·상·문학 3·4학년
- 日吉(히요시): 1·2학년 교양 + 이공 1·2년
- 矢上(야가미): 이공 3·4년·대학원
- 信濃町(시나노마치): 의·간호
- SFC(쇼난후지사와): 종합정책·환경정보 (도쿄에서 1시간+ 거리, 분리감 강함)
- 芝共立(시바쿄리츠): 약학

**다음 액션:** 사용자에게 Penmark 현실을 알리고, 풀클론 vs 빈틈공략(특히 한국인 유학생 다리 포지션) 중 선택지 제시.
