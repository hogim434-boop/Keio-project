게이오대 단일학교 한정 수업 리뷰 플랫폼 기획 (2026-04-30 ~ 2026-05-01).

**Why:** 사용자가 게이오대 재학생(한국인 추정)이고, 수업 리뷰에 특화된 플랫폼을 만들고 싶어함. 경쟁사 사전조사 완료 후 종합 기획안 작성 단계.

**핵심 시장 발견 (반드시 기억):**
- **Penmark가 게이오 학부생 80% 점유** (시간표+리뷰+커뮤니티 풀스택). 정면 경쟁 어려움
- 미나노캠퍼스가 게이오 리뷰 68,617건 보유 (양에서 절대 우위, 질에서 약점)
- 楽단リスト(keio-easylist)는 위키스타일로 운영 중
- LINE 오픈챗이 학부별·학년별로 분산되어 활발함

**2026-05-01 종합 기획안 도출 결과:**
- 서비스명 후보: **Jukulog(塾ログ)** 1순위 추천 (塾生 호칭 활용한 인사이더 네이밍)
- USP 5축: keio.jp 인증 / 5축 다차원 평가 / 시간 가중치 / 익명 Q&A / 5캠퍼스 맞춤
- MVP 5기능: 메일인증, 5축 리뷰, 캠퍼스별 뷰, 시간 가중치, 익명 Q&A
- 출시 타이밍 권장: **2026-09 가을학기 수강신청 직전**
- 페르소나 3개: 신입생 / SFC 진로고민 / 한국인 유학생
- KPI 12개월 목표: 가입 5,000명, 리뷰 5,000건, MAU 2,500

**3가지 방향 (사용자 결정 대기):**
- A안: 일본인 게이오생 정면 경쟁 (난이도 상)
- B안: 한국인 유학생 가교 포지션 (블루오션, 작은 시장)
- **C안: 정면 도전 + 차별화 (메인 추천)**

**How to apply:**
- 단순 풀클론은 어려움. 차별화 5축(인증·다차원·시간·Q&A·캠퍼스별) 유지 필수
- "정보 양"으로 미나노캠퍼스 못 이김. "질·신뢰성"으로 차별화
- SEO가 핵심 트래픽 (Penmark는 SPA로 SEO 약함, 우리의 빈틈)
- 단기 수익화 금지, 1년차 MAU 5,000 올인
- 명예훼손 리스크 최우선 대응 필요 (일본 刑法 230조)

**다음 액션 후보:**
- A/B/C안 결정
- 서비스명 확정
- 와이어프레임/PRD/DB스키마 중 다음 단계 선택
- 공동 창업자 모집 검토
