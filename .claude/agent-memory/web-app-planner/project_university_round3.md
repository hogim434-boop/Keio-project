1. 썸 온도계 (연애, ★2) — 카톡 패턴 5문항 → 썸 온도 시각화

3. 롤 멘탈 케어 봇 (게이밍, ★2) — 한 판 더 vs 그만 신호등
4. 지박령 자가진단 (멘탈, ★2) — 방콕 레벨 자조 게이지
5. 후회 가계부 (금융, ★2) — 후회 소비만 기록하는 역방향 가계부
6. 망한 일기 (창작/멘탈, ★2) — 오늘 망한 거만 적는 익명 피드

8. 술자리 룰렛 마스터 (술/놀이, ★1) — 술게임·벌칙·진실게임 통합

10. 익명 자랑·하소연 보드 (소셜, ★3) — 자랑 vs 하소연 분리 익명게시판
11. 나만의 캐릭터 일기 (창작, ★3) — 도트 캐릭터로 기분 기록
12. 즉흥 KTX 룰렛 (여행, ★3) — 랜덤 도시+코스 즉흥 추천
13. 내 인생 OST 만들기 (창작/음악, ★3) — 학기별 개인 앨범 생성
14. 이별 처방전 (연애, ★2) — 이별 시간축 회복 가이드
15. 밈 번역기 (SNS/유머, ★2) — 세대 간 신조어 번역
