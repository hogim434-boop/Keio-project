2026-04-30 두 번째 라운드. 사용자가 1차 라운드(7개: 팀플조각·공강한입·다글로 그림자·동아리살림·룸살이·자소서 거울·학기 가드)를 폐기하고 "완전히 새로운 결"을 요청.

**Why**: 1차는 학업/팀플/자취/취준만 다뤄 결이 좁았음. 사용자가 명시적으로 연애·게이밍·건강·금융·취미·여행·쇼핑·콘텐츠·술·펫·라이프스타일 등 폭넓은 카테고리 요구.

**제시한 15개 (간략)**:
1. 썸타임라인 (관계 추적)
2. 학점 도박 (친구 베팅)
3. 술자리 매니저
4. 캠퍼스 길고양이 도감
5. 디코 서버 부동산
6. 인스스 보정 챌린지
7. 푼돈 모험
8. 수업 빙고
9. 1만원 챌린지
10. 노동요 라이브
11. 안 가면 1만원
12. 책 잔향
13. 사이드 프로젝트 자랑방
14. MBTI 사주
15. 기숙사 식단 사진관

**TOP 3 (재미)**: MBTI 사주, 수업 빙고, 길고양이 도감
**TOP 3 (시장성)**: 기숙사 식단 사진관, 디코 서버 부동산, 책 잔향
**TOP 3 (초보 쉬움)**: 길고양이 도감, 1만원 챌린지, 사이드 프로젝트 자랑방

**How to apply**:
- 다음 라운드는 사용자가 좁힌 결로 PRD/와이어프레임 진입
- 사용자가 "이 결만 더 변주" 요청 가능성 있음 (예: 연애만, 진단형만)
- 1차 7개는 폐기 상태 — 다시 제안 금지
- 학교 단위 폐쇄형 / 인스타 스토리 바이럴 / 매일 재방문 후크가 이번 라운드의 강점 패턴
