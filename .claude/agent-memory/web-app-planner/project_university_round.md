2026-05-01 라운드에서 타겟이 일반 Z세대에서 "한국 대학생"으로 좁혀짐. 사용자 요청은 페르소나 분석 + 트렌드 기반 아이디어 5~7개 + 우선순위 + 추가 질문.

**검색으로 확인된 강자 영역(빈틈 좁음, 정면 충돌 비추천)**:
- 종합 학사: 에브리타임 (MAU·체류시간 1위, 사실상 표준)
- 시간표·일정: 캘박, 카카오워크 캘린더
- 강의 녹음·요약 AI: 클로바노트(20대 MAU 35%), 다글로(20대 MAU 947% 증가, 30%)
- 자소서·면접 AI: 나임, 하이잡, 사람인 AI코칭, 잡메이커, 에듀스
- 동아리·대외활동: 캠퍼스픽, 링커리어, 모두의동아리
- 팀플 일정조율: 위팀(weteam) — "언제보까" 기능 보유
- 대학생 소개팅: 연픽, 캠퍼스팅, 썸타임, 유니브팅
- 캠퍼스 중고: 파라바라(파라박스 자판기), 당근

**확인된 페인포인트 데이터**:
- 한국 청년 42.1% 취업을 "좁은 문"으로 평가
- 구직 스트레스 요인: 불확실한 미래(45.2%), 반복되는 탈락(22.1%)
- 강의실에서 AI 받아쓰기·요약을 켜놓고 딴짓하는 행태 보도(2025.11 한경)
- 노션 템플릿 선호 대학생 증가

**기획 시 빈틈 가설(검색 기반)**:
- 팀플 "기여도 가시화"는 위팀도 안 다룸
- 동아리 "내부 운영 도구"는 캠퍼스픽·링커리어 모두 모집 단계만 다룸
- 자취 룸메이트 "공동 지출 + 가사분담"은 BUBOO 일부 커버, 무로그인·가벼운 버전 빈틈
- 강의 녹음 후 "복습용 능동 학습(빈칸·플래시카드)"은 다글로 퀴즈 외 약함
- 공강 시간 "지금 당장 함께 밥/카페" 라이트 매칭은 소개팅 앱과 결이 다름

**How to apply**:
- 다음 라운드는 사용자가 7개 중 1~2개 골라 PRD/와이어프레임으로 진입
- 정면충돌 영역(에타, 시간표 종합, 자소서AI 종합)은 추천하지 말 것
- 무로그인·인스타/카톡 공유 채널 가정은 여전히 유효
