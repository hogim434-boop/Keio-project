## Step 1 — 시장·환경 분석

### 1-1. 일본 인스타그램 환경 (2026년 5월 기준)

| 지표 | 수치 | 출처 / 의미 |
|------|------|-----------|
| 일본 IG 월간 액티브 사용자 (MAU) | 약 6,600만+ | LINE 다음으로 큰 SNS 풀. **거의 모든 케이오 학생이 계정 소유** |
| 20대 IG 이용률 | **78.6%** | 전 연령 평균(48.5%)을 30%p 상회 |
| 대학생 IG 이용률 (도쿄공대 2025년 신입생) | **82.6%** | X(트위터)를 처음으로 추월. **2026년 케이오 신입생도 비슷하거나 그 이상으로 추정** |
| 여대생 IG 이용률 | **91.6%** | 9할대 횡보 — 여학생은 사실상 100% |
| 남대생 IG 이용률 | 75.5% | 매년 상승 중. 남녀 격차 좁아지는 중 |
| 기업 계정 팔로우율 | IG가 **21.9%** 로 1위 | 다른 SNS 대비 **공식 계정에 거부감 가장 적음** |

**시사점 (왜 인스타 단일 채널인가)**:
- **타겟이 거의 100% 머무는 단일 채널**. X는 정보 소비, TikTok은 오락이 강하지만, 케이오 학생이 **자기 정체성·일상**을 드러내는 채널은 인스타. 즉 "塾生らしさ"를 자극하는 콘텐츠가 자연 확산되기 가장 좋은 토양.
- LINE은 닫힌 1:1·그룹 메시징이라 신규 발견(Discovery)이 일어나지 않음. → **신규 가입 유입 채널로는 부적합**.
- 1인 운영자가 콘텐츠 제작·관리·DM 회신을 모두 처리하려면 채널을 늘릴수록 품질이 떨어짐. **단일 채널 깊이 우선**이 전략적으로 옳다.

### 1-2. 케이오대 학생의 인스타 사용 특성

| 계정 유형 | 특징 | 활용 포인트 |
|----------|------|-----------|
| **공식 학생단체 계정** (예: `@mitasai_pr` 三田祭 공식, 6.9k 팔로워) | 학년·캠퍼스 가리지 않고 모두 팔로우. 학교 일정·축제 업데이트 허브 | **콜라보 1순위 타깃** — 같은 6~10k 규모로, "동급 마이크로 협업" 가능 |
| **동아리 (サークル) 계정** | 1k~5k 규모. 신입생 환영 시즌(4월)에 가장 활발 | **스토리 멘션 교환** 으로 빠르게 시드 확보 |
| **학년·학부 비공식 계정** (예: 〇〇学部25年度入学 등) | 익명 운영자가 신입생 대상 정보 큐레이션. 1k~3k | **공통 팔로잉이 두꺼움** → 한 번 노출되면 도미노 효과 |
| **개인 일상 계정 / "裏垢"** | 친한 친구 30~100명. 본심을 적는 비공개 계정 | **Jukulog 의 핵심 인사이트**: 케이오 학생들은 이미 "본심을 말할 곳"을 따로 만들고 있다 → "혼자 裏垢 만들 필요 없이 같은 塾生끼리 모인 곳" 으로 어필 |

### 1-3. 일본 대학생 익명 커뮤니티의 역사와 한계

| 서비스 | 시기 | 특징 | 종말/한계 |
|--------|------|------|----------|
| **2ch / 5ch** | 1999~ | 완전 익명 게시판 문화 원형 | 외부인·고령층 혼재, UI 낡음, 대학생에게 더 이상 매력 없음 |
| **ひま部** | 2015–2019 | 학생 한정 챗 SNS, 100만+ MAU | **연령 인증 미흡** → 50대 남성이 학생 사칭 → **18세 미만 피해 SNS 2위 (214건)** → 2019년 강제 종료 |
| **Yay!** | 2020~ | ひま部 후속, 200만+ 등록 | 다시 출회계 (出会い系) 색채 강해짐, 대학생 거리감 |
| **마시멜로 (マシュマロ)** | 2017~ | 익명 메시지 받기 도구 | 1:1 단발 — 커뮤니티 효과 없음 |

**시사점 (Jukulog 가 반드시 강조해야 할 것)**:
1. **「학생 한정 SNS = 위험하다」는 학습된 불신** 이 일본 대학생들에게 박혀 있다. 「ひま部 사건」을 기억하는 세대다.
2. 따라서 **"안전"은 부가 가치가 아니라 1차 가치 제안**. keio.jp 이메일 인증을 단순 편의가 아니라 **「당신과 같은 塾生만 들어옵니다」 라는 안심 신호** 로 포지셔닝해야 한다.
3. 5ch의 "완전 익명"은 매력적이지만 **외부인 침입**으로 더러워진다. Jukulog는 **"닫힌 익명 (クローズド匿名)"** 이라는 일본 시장에서 거의 비어 있는 슬롯에 들어간다.

### 1-4. 경쟁 포지셔닝 (PENMARK 대비)

```
                 외부인 차단 강함
                       │
          [Jukulog]    │
        (익명·잡담)    │
                       │
   ────────────────────┼──────────────────  콘텐츠 영역
                       │
        [PENMARK]      │     [LINE オープンチャット]
      (강의·시간표)    │       (외부 침입 빈번)
                       │
                  외부인 차단 약함
```

- **PENMARK**: 케이오 학부생 **80% 점유**, 강의 리뷰·시간표 관리에 강함. 본명 기반·학업 도구 색채.
- **Jukulog**: PENMARK 가 비워둔 영역인 **"학업 외 모든 본심 (恋愛, バイト愚痴, 教授の悪口, 進路の不安)"** 을 가져간다.
- **PENMARK 와 직접 충돌하지 않는다** — 오히려 **공존 가능** ("授業 보러 PENMARK, 一息 つきに Jukulog"). 이 메시지는 광고에서도 그대로 써도 됨.

> **1줄 차별점 슬로건**:
> 「**PENMARKで授業、Jukulogで本音。**」
> (PENMARK에선 수업, Jukulog 에선 본심.)
>
> *이유*: 적이 아니라 보완재로 포지셔닝해서, 이미 PENMARK에 있는 80%의 학생을 *한 번 더 다운로드시키는* 형태로 진입. 학생 머릿속에서 두 앱은 분리된 슬롯에 들어간다.

## Step 3 — 가치 제안 & 핵심 메시지

### 3-1. 슬로건 후보 3개

| 후보 | 일본어 | 한국어 의도 | 적합도 |
|------|--------|------------|--------|
| **A (메인 추천)** | **塾生だけが、本音で話せる場所。** | "塾生 (케이오 학생) 만이 본심으로 이야기할 수 있는 공간." | ★★★★★ — 「塾生」이라는 정체성 단어 + 「本音」 (본심) 의 일본 특유 무게감을 한 번에 잡음 |
| B | **慶應の中の、もうひとつの居場所。** | "케이오 안의 또 하나의 거처." | ★★★★ — 따뜻하지만 "본심" 만큼 행동을 유도하지 못함 |
| C | **keio.jp の人だけが入れる、匿名のひとり言。** | "keio.jp 인 사람만 들어올 수 있는 익명 혼잣말." | ★★★ — 기능 설명 직접적, 시 적성격 약함 |

> **메인 슬로건은 A** 「**塾生だけが、本音で話せる場所。**」.
>
> *왜 A가 다른 후보보다 나은가*: 「塾生」은 케이오 학생들이 자기 자신을 부를 때 쓰는 **인사이더 단어**. 외부인은 "慶應生" 이라고 하지 "塾生"이라고 안 한다. 첫 한 마디부터 "내 부족 사람이 만든 서비스" 라는 신호를 발신 → 신뢰 구축 비용을 0 으로 만든다.

### 3-2. 한 문장 답변: "왜 다른 SNS 두고 Jukulog?"

> **「Twitterは外、Instagramは表、Jukulogは塾生だけの裏。」**
> (Twitter는 외부, Instagram은 겉, Jukulog는 塾生만의 뒤편.)

*이유*: 학생들이 이미 "겉(表) 인스타" 와 "뒤(裏) 비공개 계정" 을 분리해서 운영한다는 행동 사실에 올라탄 메시지. 새 행동을 강요하지 않고 **"裏垢"의 더 안전한 대체재** 라는 익숙한 슬롯에 끼워 넣음.

> **이유**: 익명 SNS 의 첫 진입 장벽은 **"안전한가?"**. 안전 → 가치 → 차별 → 비용 순서로 쌓아야 회의적인 일본 학생의 마음이 움직인다. 한국 시장이라면 "익명 → 재미 → 안전" 순서로도 가지만, 일본은 **신뢰 → 가치** 순서가 표준.
