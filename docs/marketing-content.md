### A-2. Bio 후보 3개

> 인스타 Bio 한도 = 150자 이내. 줄바꿈은 「Notes 앱에 작성 → 복사 → 인스타에 붙여넣기」 순서로 하면 깨지지 않습니다 (인스타 앱에서 직접 줄바꿈 입력 시 가끔 한 줄로 합쳐지는 버그 있음).

#### Bio 안 ① — 「안전·신뢰 우선형」 (★ 채택 추천)

```
塾ログ｜慶應生だけの匿名コミュニティ
🔵 keio.jp 認証で外部完全シャットアウト
🔴 三田・日吉・SFC キャンパス別ギャラリー
💬 授業の愚痴も、就活の本音も。
👇 招待 & ログインはこちら
```

- **글자수**: 약 95자 (이모지 4개 제외)
- **이런 페르소나에게 통한다**: 페르소나 1 (日吉ぼっち) + 페르소나 2 (三田 의식 높은 系). 첫 줄 「慶應生だけ」 로 안전 → 두 번째 줄 인증 시스템으로 ひま部 학습된 불신 차단 → 세 번째 줄로 차별화 → 마지막에 행동 유도. **메모리에 기록된 일본 시장 메시지 우선순위 (신뢰 40 → 가치 35 → 차별 15 → 비용 10) 정확 일치**.

#### Bio 안 ② — 「인사이더 톤형」

```
塾生だけが、本音で話せる場所。
🎓 keio.jp / sfc.keio.ac.jp 認証制
✏️ 授業・就活・サークル・恋バナ全部OK
🌸 中の人がコッソリ運営中
👇 ログインはここから
```

- **글자수**: 약 85자
- **이런 페르소나에게 통한다**: 페르소나 4 (内部進学リア充) + 페르소나 3 (SFC おたく). 「塾生」 인사이더 단어 + 「中の人」 운영자 친근체 → "이건 우리 부족 사람이 만들었다" 신호. 다만 **첫 줄에서 안전 메시지보다 정체성을 먼저** 제시하므로 ひま部 트라우마 강한 사용자에게는 ① 보다 약함.

#### Bio 안 ③ — 「기능 직설형」

```
慶應生限定の匿名掲示板アプリ
✅ keio.jp ログイン必須
✅ 完全匿名・スクショ通報あり
✅ 三田/日吉/SFC 別カテゴリ
🆓 完全無料・広告なし
🔗 はじめる
```

- **글자수**: 약 80자
- **이런 페르소나에게 통한다**: 페르소나 5 (文学部院生) — 정중하고 신중한 정보 수집형. 체크 마크 직렬은 신뢰성·기능 정보 동시 전달에 효과적. 다만 톤이 마이크로소프트 제품 카피 같아 **20대 학생 감성에는 차갑게 느껴질 위험**.

> **결정**: **Bio ① 채택**. 이유는 (1) 메시지 우선순위 정확 일치 (2) 이모지 색상으로 케이오 블루·레드 즉시 인식 (3) 「授業の愚痴も、就活の本音も」 한 줄로 5인 페르소나 모두 포섭.

### A-3. 프로필 사진 컨셉 3개

> 운영자가 Canva 무료 플랜에서 **15분 안에 만들 수 있는 수준** 으로 구체화했습니다. 다 만들고 셋 중 인스타 스토리에 올려 폴 (アンケート) 로 24시간 사용자 의견 받아도 좋습니다.

#### 컨셉 ① — 「ペン+塾」 미니멀 로고 (★ 채택 추천)

| 항목 | 사양 |
|------|------|
| 캔버스 | 1080×1080px (인스타 프로필 사진은 320×320 으로 표시되지만, 큰 사이즈로 만들어야 또렷) |
| 배경색 | 케이오 블루 `#002B5C` 단색 |
| 메인 요소 | 흰색 「塾」 한 글자, 폰트: **Noto Serif JP Bold**, 크기: 캔버스 50% (약 540pt) |
| 서브 요소 | 「塾」 의 좌상단에 작은 흰색 만년필(ペン) 아이콘 (Canva 무료 아이콘 「fountain pen white」 검색) |
| 톤 | 학술적·정통 (慶應義塾 의 「義塾」 한자 분위기 계승) |

**왜 이게 효과적인가**: 케이오 블루 + 「塾」 한자 한 글자는 **케이오 학생이 0.3초 안에 인지** 하는 시각 코드입니다. 부족 식별 신호로 가장 강력. 페르소나 5인 모두에게 동일하게 작동.

#### 컨셉 ② — 「JL」 모노그램

| 항목 | 사양 |
|------|------|
| 배경색 | 흰색 `#FFFFFF` |
| 메인 요소 | 「JL」 두 글자 모노그램, 폰트: **DM Serif Display Bold**, 색상 케이오 블루 `#002B5C` |
| 서브 요소 | 「JL」 아래에 작게 「JUKULOG」 캐피탈 폰트 (Inter Bold 14pt 정도, 그레이 `#666`) |
| 톤 | 모던·세련 (Brandy / Patreon 풍) |

**왜 이게 효과적인가**: 영문 모노그램은 **인스타 그리드에서 다른 일본어 계정과 시각적으로 구별** 됩니다. 단점은 「塾」 한자가 빠져 케이오 시그널이 약함 — 페르소나 3 (SFC) 에게는 잘 맞지만 페르소나 4 (内部進学) 에게는 약간 차가움.

#### 컨셉 ③ — 「中の人」 일러스트

| 항목 | 사양 |
|------|------|
| 배경색 | 베이지 `#F5EDE0` |
| 메인 요소 | 노트북 보고 있는 학생 일러스트 (Canva 무료 일러스트 「student laptop minimal」). 얼굴은 보이지 않고 머리·후드 뒷모습 |
| 서브 요소 | 노트북 화면에 작게 「塾ログ」 글자 |
| 톤 | 친근·익명 (운영자 = 같은 학생 신호) |

**왜 이게 효과적인가**: 「中の人」 운영 톤과 시각이 일치 → 얼굴 안 보이는 익명성 + 같은 학생이라는 친밀도. 다만 **공식 계정 무게감이 약해 PENMARK 같은 공식 서비스 옆에 놓이면 비공식 인상**. 따라서 **W1 ~ W4 까지는 컨셉 ①, M3 (정착) 시점에 ③ 으로 교체** 하는 2단계 운영 추천.

> **결정**: **컨셉 ① 「ペン+塾」 채택**. M3 시점에 사용자 1,000명 돌파하면 컨셉 ③ 일러스트 안으로 교체 검토 (그때는 익명 친밀도가 신뢰보다 중요해지는 단계).

### A-4. 하이라이트 카테고리 5개

> 인스타 하이라이트 커버 = **단색 배경 + 흰 글자 1단어** 가 일본 인스타에서 가장 검증된 패턴 (PENMARK, 전국 대학 공식 계정 80% 이상 동일 형식). 운영자가 Canva 에서 1080×1920 (스토리 사이즈) 로 만들고, 인스타 스토리에 비공개 업로드 → 해당 스토리를 하이라이트 커버로 지정하는 방식입니다.

| # | 라벨 (6자 이내) | 커버 디자인 | 들어갈 스토리 종류 |
|---|---------------|-----------|------------------|
| 1 | **はじめに** | 케이오 블루 `#002B5C` 배경 + 흰색 「📖」 이모지 + 그 아래 흰 「Noto Sans JP Bold 80pt」 로 「はじめに」 | 가입 방법 4컷 / FAQ / 인증 도메인 안내 / 「※スクショ拡散はガイドライン違反」 워터마크 안내 |
| 2 | **塾生の声** | 핑크 `#E91E63` 배경 + 흰색 「💬」 이모지 + 「塾生の声」 | 사용자 후기 캡처 (익명·이름 마스킹). W2 부터 모이는 후기 적재 |
| 3 | **キャンパス** | 그라데이션 (`#002B5C` → `#E91E63` → `#FFA000` 좌우 횡단) + 흰색 「🏛」 + 「キャンパス」 | 三田/日吉/SFC 별 인기 글 모음. 캠퍼스 사진 + 인용문 슬라이드 |
| 4 | **就活・学業** | 검정 `#111111` 배경 + 흰색 「📚」 + 「就活・学業」 | 시즌 콘텐츠 아카이브 (시험·취업·三田祭 카운트다운 등) |
| 5 | **運営から** | 그레이 `#444444` 배경 + 흰색 「✏️」 + 「運営から」 | 업데이트 공지 / 운영자 일기 / 로드맵 / 사과문 (있으면) |

**왜 이 5개·이 순서인가**:
- 1번 「はじめに」 가 맨 왼쪽 = 인스타 프로필 진입 후 가장 먼저 눈에 띄는 위치. 신규 방문자 가입 동선 0클릭 단축.
- 2번 「塾生の声」 가 두 번째 = 일본 사용자가 가입 결정 전 가장 보고 싶어하는 「소셜 프루프」.
- 5번 「運営から」 가 마지막 = 운영 투명성 신호 (일본 사용자는 「운영자가 보이는」 것을 매우 중요시).
