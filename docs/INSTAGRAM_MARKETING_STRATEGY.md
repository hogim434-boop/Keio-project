# 塾ログ (Jukulog) Instagramマーケティング戦略書

> 慶應義塾大学生限定・匿名掲示板サービス「塾ログ」の公式Instagram運用戦略
> 作成日: 2026-05-07 / 対象期間: ローンチ〜12ヶ月
> 想定運用体制: 1人運営 + 学生アンバサダー数名(無償〜謝礼程度)

---

## 1. エグゼクティブサマリー

- **戦略の核心**: 「慶應生のリアルな本音」をInstagramリールで切り取り、"あるある共感 → プロフィール訪問 → @keio.jpメール認証で登録"の3ステップ動線を磨き込む。リール7割・ストーリーズ2割・フィード1割の配分で運用負荷を抑えつつ、新学期(4月)・三田祭(11月)・就活解禁(3月)の3大ピークに合わせて山を作る。
- **差別化ポイント**: PENMARKやUnipa等の「実名×時間割管理」系SNSと真っ向勝負せず、「**塾生だけの裏アカ的空間**」というポジションで、連絡先交換を強制されない"逃げ場"として打ち出す。匿名性を「怖い」ではなく「**素でいられる場所**」として再定義する。
- **数値目標との接続**: 3ヶ月でフォロワー1,500人 → 登録300人(CVR 20%)、12ヶ月でフォロワー8,000人 → 登録2,000人(CVR 25%)。広告予算は月¥0〜¥30,000の範囲で、リール拡散と学内アンバサダー紹介でオーガニック中心に成長させる。

### 2-3. 共通インサイト(マーケ上の武器)

1. **「慶應である」というアイデンティティへの誇りと、それを公言することの恥ずかしさが共存** → 匿名で本音を言える場の需要は確実にある

4. **PENMARK等への不満**: 実名・時間割晒しがしんどい。"逃げ場"としての匿名空間ニーズ

## 13. リスク管理

### 13-1. 「慶應」名称使用に関するリスク

| リスク | 対応策 |
|-------|-------|
| 大学公認と誤認される | プロフィール冒頭に「※本サービスは慶應義塾大学公認ではありません」を明記 |
| 慶應ロゴ・校章使用NG | 校章・公式ロゴは一切使用せず、独自デザインで「塾ログ」ブランディング |
| 「慶應」商標問題 | サービス名は「塾ログ」とし、「慶應」は説明文中での使用に留める |
| 大学から警告を受ける可能性 | 事前に学生課に相談、もしくは弁護士に商標確認を依頼(¥30,000程度) |

### 13-2. 匿名コミュニティのネガティブイメージ対応

| 想定される批判 | 先回り対応策 |

| 「親に怒られそう」 | 「保護者の方へ」FAQページを用意、安全性をPR |
